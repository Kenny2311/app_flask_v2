Recomendaciones basadas en el modelo
Campos financieros clave para el modelo
  • Ingreso Anual / Salario Mensual: Asegúrese de ingresar valores consistentes; el modelo los usa para calcular ratios de endeudamiento.
  • Deuda Pendiente: Incluya todas las deudas activas; valores subestimados pueden afectar la predicción de riesgo.
  • Ratio Deuda/Ingreso: Revise que esté correctamente calculado; es uno de los indicadores más importantes para la predicción.
  • Ratio EMI/Ingreso: Indica carga mensual de préstamos; valores coherentes ayudan a que el modelo identifique riesgo de default.
  • EMI Total Mensual: Refleja compromisos financieros mensuales; datos incorrectos reducen la precisión.
  • Monto Invertido Mensualmente / Ahorro Mensual: Ayuda al modelo a distinguir estabilidad financiera.
Historial de crédito y consultas
  • Nro Consultas Crédito: Incluya todas las consultas recientes; el modelo lo usa para detectar riesgo de sobreendeudamiento.
  • Antiguedad Historial Crédito: Años de historial; más histórico ayuda al modelo a diferenciar solicitantes confiables.
  • Infocorp Flag: Indica si el solicitante tiene registros negativos; información correcta es crucial para evaluar riesgo.
Otros campos relevantes para el modelo
  • Nro Cuentas Bancarias / Nro Tarjetas Crédito: El modelo utiliza estos valores para estimar exposición financiera.
  • Tasa Interes / Tipo Prestamo: Información precisa sobre tasas y tipo de préstamos ayuda a predecir riesgo.
  • Cambio Limite Credito: Refleja cambios recientes en disponibilidad de crédito; puede indicar riesgo emergente.
  • Sector Económico / Region: Contextualiza el comportamiento financiero según actividad y ubicación.
